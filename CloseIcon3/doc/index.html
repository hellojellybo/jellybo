Getting Started

CloseIcon3 - is simple jQuery plugin that provides animated close icon. Close Icon animate on mouse over. It’s well documented and easy to integrate in your project.

Integration

First include required libraries:

jQuery >= 1.8 lib/jquery/jquery.js
Snap.svg >= 0.5.1 lib/snap/snap.svg-min.js
JellyboBase - the latest version lib/jellybo/jellybo-base.min.js

Include all libraries only once even if you use more animations.
Then include animation file:

js/animation.min.js

For any errors in integration check the console !

Initialization

Set class jellybo-CloseIcon003 to the element

<div class="jellybo-CloseIcon003"></div>

or use the jQuery plugin

<script>
$(document).ready(function (){
    $('#elem').jellyboCloseIcon003();

})
</script>

Setting configuration

Passing as data-* attributes

<div class="jellybo-CloseIcon003" data-width="50"></div>

or passing object to the jQuery plugin

<script>
$(document).ready(function (){
    $('#elem').jellyboCloseIcon003({
        width: 500,
    });

})
</script>

Configuration

Attributes

size size of the icon
color color of the icon
stroke width of the stroke
