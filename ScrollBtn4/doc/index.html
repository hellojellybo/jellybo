Getting Started

ScrollButton004 - is simple jQuery plugin that provides animated scroll button. Scroll button indicates that page will continue after scrolling. It’s well documented and easy to integrate in your project.

Required libraries

css style assets/css/scrollbutton.css
fonts assets/css/fonts (must be in the same folder as scrollbutton.css)
jquery assets/lib/jquery/jquery.js
snap svg assets/lib/snap/snap.svg-min.js
jellybo scroll button assets/js/scrollbutton.js

Initialization

Set class jellybo-ScrollBar004 to the element

<div class="jellybo-ScrollBar004"></div>

or use the jQuery plugin

<script>
$(document).ready(function (){
    $('#searchBtn1').jellyboScrollButton004();

})
</script>

Setting configuration

Passing as data-* attributes

<div class="jellybo-ScrollBar004" data-width="500"></div>

or passing object to the jQuery plugin

<script>
$(document).ready(function (){
    $('#searchBtn1').jellyboScrollButton004({
        width: 500,
    });

})
</script>

Configuration

Attributes

width width of the icon
height height of the icon
color1 color of moving border line and dot in the middle
color2 color of static border line
shadow_color color of the shadow
text the "Scroll" text
loops number of loops before longer break (default: 1)
line_stroke width of stroke
